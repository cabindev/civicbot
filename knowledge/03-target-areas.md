# พื้นที่เป้าหมาย (10 ประเด็นหลัก)

แผนงาน มี.ค. 2568 – มี.ค. 2570

| จังหวัด | ประเด็น/แกนเรื่อง |
|---|---|
| ศรีสะเกษ | เมืองบูรณาการลดปัจจัยเสี่ยง |
| อ่างทอง | — |
| ตาก | กระทงสายสุขใจ |
| สุรินทร์ | งานช้างสุรินทร์ |
| น่าน | ทำไมน่านดื่มเหล้าอันดับหนึ่ง / แข่งเรือน่าน (แกนจริงมักเป็นเรื่องเด็กน่าน/เยาวชน โดยมีงานประเพณีเป็นฉากหนึ่ง) |
| เชียงราย | เยาวชน |
| แม่ฮ่องสอน | วิถีวัฒนธรรม |
| นครศรีธรรมราช | บุญสารทเดือนสิบ |
| สุโขทัย | ลอยกระทงเผาเทียนเล่นไฟ |
| ขอนแก่น | สงกรานต์ถนนข้าวเหนียว |
| พะเยา | แม่ใจ อำเภอลดปัจจัยเสี่ยง |

> หมายเหตุ: รายละเอียดเฉพาะของแต่ละโปรเจคให้ยึดตามเอกสารจักรวาลล่าสุดของทีม หากไม่มีข้อมูลในไฟล์นี้ ให้บอกว่าต้องตรวจสอบกับทีมเพิ่ม
