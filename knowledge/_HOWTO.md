# วิธีเพิ่ม/แก้ฐานความรู้ของบอท

โฟลเดอร์นี้คือ "สมอง" ของบอท — ทุกไฟล์ `.md` ที่นี่จะถูกโหลดรวมเข้า system prompt
ของ Gemini โดยอัตโนมัติทุกครั้งที่บอทตอบ **ไม่ต้องแก้โค้ดเลย**

## เพิ่มประเด็น/จักรวาลใหม่
1. สร้างไฟล์ `.md` ใหม่ในโฟลเดอร์นี้ ตั้งชื่อขึ้นต้นด้วยเลขเพื่อคุมลำดับ เช่น
   `06-issue-soft-power.md`, `07-area-nan.md`
2. เขียนเนื้อหาเป็น Markdown ตามปกติ (หัวข้อ ตาราง bullet ได้หมด)
3. บันทึก แล้วรีสตาร์ทแอป — บอทจะรู้เรื่องใหม่ทันที

## กฎการเขียนที่ควรทำตาม
- **เขียนเป็นข้อเท็จจริงที่ตรวจสอบแล้ว** — บอทจะเชื่อทุกอย่างในไฟล์นี้
- **ตัวเลข/สถิติใส่ที่มาเสมอ** ถ้ายังไม่ชัวร์ อย่าใส่ หรือเขียนกำกับว่า "(ต้องตรวจสอบ)"
- หนึ่งไฟล์ = หนึ่งประเด็น/หนึ่งจักรวาล จะได้จัดการง่าย
- ไฟล์ที่ขึ้นต้นด้วย `_` (เช่นไฟล์นี้) จะ **ไม่** ถูกโหลดเข้า prompt ใช้สำหรับโน้ตภายใน

## หมายเหตุ
- โทน/บุคลิก/กฎเหล็กของบอท อยู่ใน `config.py` (`SYSTEM_PROMPT`) ไม่ใช่ที่นี่ — ที่นี่เก็บเฉพาะ "ความรู้/ข้อเท็จจริง"
- เอกสารสเปกภาพรวมอยู่ที่ `BOT.md`
